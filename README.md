# Healthcare Management System

## 📌 Overview

A full-stack healthcare web application designed to securely manage and share patient medical records among doctors, patients, and pharmacies. The system uses role-based authentication and REST-style APIs to ensure secure and controlled access to sensitive healthcare data.

## 🚀 Features

* Role-based login system (Doctor, Patient, Pharmacy)
* Secure medical record management
* RESTful API-based backend communication
* QR-code encrypted access for record sharing
* Authentication and authorization workflows
* Reliable transaction handling with layered architecture

## 🛠 Tech Stack

* **Backend:** Java (Servlets / REST APIs)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MySQL
* **Architecture:** Layered Architecture (Controller → Service → DAO)

## 🔐 Security Highlights

* Role-based access control (RBAC)
* Encrypted QR-based data sharing
* Secure authentication workflows
* Structured backend validation
